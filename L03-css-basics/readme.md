# Lecture 03 – CSS Basics

## What I implemented this lecture
- Added external CSS to style my portfolio page without altering the core HTML structure.
- Implemented professional spacing using margins and padding to create a readable layout constrained by max-width.
- Styled a horizontal navigation menu by removing default list bullets and link underlines.

## CSS selectors I used (at least 5)
- Element selector (`body`, `h1`, `img`)
- Group selector (`header, footer`)
- Descendant selector (`nav ul`, `nav li`)
- ID selector (`#main`)
- Pseudo-class (`a:hover`)

## One thing I struggled with
- Understanding when it is best to use margin versus when to use padding to create spacing between elements.
