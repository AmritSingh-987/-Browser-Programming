Implemented Features
Skills Section: A new section displaying 6 technical skills (HTML, CSS, JS, Python, IoT, and Network Security) in a clean grid.

Projects Section: A section featuring 2 projects with titles, descriptions, and links.

Theme Toggle: A button that switches the page between Light and Dark modes using a state variable and classList.

Theme Persistence: User theme preference is saved to and read from localStorage using the key portfolio_theme.

Automated Timestamp: A "Last updated" string in the footer that automatically generates the current date in YYYY-MM-DD format upon page load.

How to Test
Skills/Projects: Scroll down to view the newly added sections.

Theme: Click the Toggle Theme button at the top. Refresh the page to confirm your choice was saved.

Date: Look at the bottom of the page to see the automatically generated "Last updated" date.
